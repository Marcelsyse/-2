# Random Quote Generator

**Автор:** Владислав Шарафутдинов

## Описание

Приложение на Python с графическим интерфейсом для генерации случайных цитат. Позволяет:
- Генерировать случайную цитату из предустановленного списка
- Фильтровать цитаты по автору и теме
- Добавлять новые цитаты (с проверкой на пустые строки)
- Сохранять историю сгенерированных цитат в JSON-файл
- Загружать историю при запуске

## Требования

- Python 3.6 или выше
- Стандартные библиотеки Python (tkinter, json, os, random, datetime)
